\subsection{Detailed Description}
Drawing Open\-G\-L E\-S primitives.
\begin{DoxyItemize}
\item cc\-Draw\-Point
\item cc\-Draw\-Line
\item cc\-Draw\-Poly
\item cc\-Draw\-Circle
\item cc\-Draw\-Quad\-Bezier
\item cc\-Draw\-Cubic\-Bezier
\end{DoxyItemize}

You can change the color, width and other property by calling the gl\-Color4ub(), gl\-Line\-Width(), gl\-Point\-Size().

\begin{DoxyWarning}{Warning}
These functions draws the Line, Point, Polygon, immediately. They aren't batched. If you are going to make a game that depends on these primitives, I suggest creating a batch. 
\end{DoxyWarning}
