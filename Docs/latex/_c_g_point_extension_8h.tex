\subsection{Detailed Description}
C\-G\-Point extensions based on Chipmunk's cp\-Vect file. These extensions work both with C\-G\-Point and cp\-Vect.

The \char`\"{}ccp\char`\"{} prefix means\-: \char`\"{}\-Co\-Cos2d Point\char`\"{}

Examples\-:
\begin{DoxyItemize}
\item ccp\-Add( \hyperlink{_c_g_point_extension_8h_a6250be5f1c27162404561148ebd6ffcc}{ccp(1,1)}, \hyperlink{_c_g_point_extension_8h_a6250be5f1c27162404561148ebd6ffcc}{ccp(2,2)} ); // preferred cocos2d way
\item ccp\-Add( C\-G\-Point\-Make(1,1), C\-G\-Point\-Make(2,2) ); // also ok but more verbose
\end{DoxyItemize}


\begin{DoxyItemize}
\item cpvadd( cpv(1,1), cpv(2,2) ); // way of the chipmunk
\item ccp\-Add( cpv(1,1), cpv(2,2) ); // mixing chipmunk and cocos2d (avoid)
\item cpvadd( C\-G\-Point\-Make(1,1), C\-G\-Point\-Make(2,2) ); // mixing chipmunk and C\-G (avoid) 
\end{DoxyItemize}
