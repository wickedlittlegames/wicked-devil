Detailed Description

Object which may be used to embed a latitude / longitude point as the value for a key in a PFObject. PFObjects with a PFGeoPoint field may be queried in a geospatial manner using PFQuery's whereKey:nearGeoPoint:.

This is also used as a point specifier for whereKey:nearGeoPoint: queries.

Currently, object classes may only have one key associated with a GeoPoint type.
