28  @discussion
29  A common use of an PF_FBFriendPickerCacheDescriptor instance, is to allocate an instance
30  at the point when a session is opened, and then call prefetchAndCacheForSession. This

32  If at some point the user goes to select friends, the PF_FBFriendPickerViewController
